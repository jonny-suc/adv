2. 版本规划¶

2.1. 产品定位¶

  1. 对全站广告进行管理，包括PC、M、MIP、APP端（源生页、H5页）。

  2. 同时提供给内部运营及外部合作人员使用。

  3. 根据项目分配管理者，项目管理者能权限管理广告。整站后台管理员负责审核广告。

  4. 提供外部接口广告输出。

  5. 提供数据跟踪及可视化图表

2.3. 定义用户行为¶

用户行为!¶

行为

用户群体

具像

备注

管理广告项目

管理员

创建“盛耀广告”项目

包括创建、改名、更改分配权限、删除

分配权限

管理员

分配“盛耀广告”项目权限到用户A中

管理广告位

管理员，项目管理员

在“盛耀广告”项目中创建“排行榜右侧医生”广告位

包括创建、改名、删除

管理广告

管理员，项目管理员

“排行榜右侧医生”广告位创建广告“罗浩医生”

包括创建、改名、删除、上下架

完成广告设置

管理员，项目管理员

  1. 选择广告形式“图片”

  2. 完成上传图片

  3. 选择上架时间2019/6/1-2019/7/1

  4. 选择优先级：0

  5. 填写落地页：www.baidu.com

  6. 指定显示范围：糖尿病

  7. 指定用户范围：男性/不限地区/不限年龄

提交审核

管理员，项目管理员

提交审核

审核广告

管理员

审核不通过，填写原因：素材太丑

包括通过、不通过
